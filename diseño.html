DISEÑO DEL BRAZO ROBÓTICO

Galeria del diseño

En primer lugar, se generó un modelo 3D del brazo robótico controlable de forma dinámica por Simscape y Robotics System Toolbox utilizando el modelo de SolidWorks de este. Al haberse generado se compiló un programa Simulink que puede ser llamado por medio de un script que se encarga de generar las variables necesarias para poder ser utilizado dentro de un bloque de control cinemático.
